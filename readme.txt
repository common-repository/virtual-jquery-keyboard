== Description ==
Adds a JQuery Keyboard to input fields using a CSS class. Works with any contact form plugin! Contact Form 7, Gravity Forms, Caldera Forms, Ninja Forms

Currently this plugin is limited to [ML Keyboard](http://mbut.github.io/jquery.mlkeyboard/)

You can add the CSS class `jquery-keyboard` to any `input` tag to activate the keyboard.
